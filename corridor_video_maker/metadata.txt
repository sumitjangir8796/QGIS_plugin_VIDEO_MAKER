about=Corridor Video Maker animates the QGIS 2D map canvas along any polyline centerline to produce an MP4 fly-through video. At each frame the view centres on the current position and auto-rotates so the travel direction always faces straight up, giving a clean forward-facing corridor perspective. Features: click-on-map start point selection, configurable travel speed / FPS / corridor buffer width / output resolution, Gaussian turn smoothing, distance progress bar overlay with start/end labels, and off-screen rendering (the live canvas is never disturbed). Requires opencv-python (auto-installed on first use).
